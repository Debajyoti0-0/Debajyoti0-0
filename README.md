## ⚡ Featured Projects

** [ToriFY](https://github.com/Debajyoti0-0/ToriFY) - **106 stars + 11 forks** 
  A tool to automatically anonymize traffic by redirecting all network traffic through the TOR network, rotating IPs at configurable intervals, and randomizing the MAC address.
  **Tech Stack:** `Python`, `Tor`, `Macchanger`

** [AirHunt](https://github.com/Debajyoti0-0/AirHunt) - **71 stars + 12 forks** 
  A versatile wireless network hacking toolkit enabling monitor mode, handshake capture, deauth attacks, and WPS exploitation for comprehensive wireless security assessments.
  **Tech Stack:** `Python`, `Aircrack-ng`, `Reaver`, `Bully`

** [Brahmastra](https://github.com/Debajyoti0-0/Brahmastra) - **17 stars + 8 forks** 
  A comprehensive security analysis tool inspired by ancient Indian mythology. Combines OSINT, network scanning, web crawling, CMS detection, SQLi/XSS scanning, and SSL analysis into a powerful offensive security toolkit.
  **Tech Stack:** `Python`, `Nikto`, `Nmap`, `WPScan`, `ffuf`

** [TriNetra](https://github.com/Debajyoti0-0/TriNetra) - **2 stars + 1 fork** 
  "The Third Eye that Sees Beyond the Surface." A fast, multi-threaded web reconnaissance crawler designed to uncover hidden endpoints, API keys, and JWTs with Tor support and rich CLI output.
  **Tech Stack:** `Python`, `Rich CLI`, `httpx`

** [No4bidden](https://github.com/Debajyoti0-0/No4bidden) - **2 stars + 2 forks** 
  An advanced HTTP 40X (Forbidden/Unauthorized) bypass scanner that identifies misconfigurations allowing unauthorized access to restricted resources.
  **Tech Stack:** `Python`, `Requests Library`

** [DomainShield](https://github.com/Debajyoti0-0/DomainShield) - **1 fork** 
  An automated security assessment tool that performs comprehensive DNS analysis, email authentication validation (SPF/DKIM/DMARC), and network security scanning, generating professional HTML reports with actionable findings.
  **Tech Stack:** `Shell Scripting`, `OpenSSL`, `dig`
