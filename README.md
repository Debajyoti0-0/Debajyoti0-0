## ⚙️ GitHub Actions — Metrics Setup

> **Drop this workflow into `.github/workflows/metrics.yml`** in your profile repository (`Debajyoti0-0/Debajyoti0-0`) to auto-generate live metric SVGs.

```yaml
name: Metrics
on:
  schedule:
    - cron: "0 */6 * * *"   # Refresh every 6 hours
  workflow_dispatch:
  push:
    branches: ["main"]

jobs:
  github-metrics:
    runs-on: ubuntu-latest
    permissions:
      contents: write

    steps:
      # ── Base Stats + Language Activity ──────────────────────────────────────
      - name: Base Metrics
        uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          filename: metrics.base.svg
          base: header, activity, community, repositories, metadata
          plugin_languages: yes
          plugin_languages_indepth: yes
          plugin_languages_details: lines, bytes-size, percentage
          plugin_lines: yes
          plugin_activity: yes
          plugin_activity_limit: 5
          plugin_activity_visibility: public
          config_timezone: Asia/Kolkata

      # ── Achievements ────────────────────────────────────────────────────────
      - name: Achievements
        uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          filename: metrics.achievements.svg
          base: ""
          plugin_achievements: yes
          plugin_achievements_threshold: B
          plugin_achievements_secrets: yes
          plugin_achievements_display: compact
          config_timezone: Asia/Kolkata

      # ── Coding Habits ────────────────────────────────────────────────────────
      - name: Coding Habits
        uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          filename: metrics.habits.svg
          base: ""
          plugin_habits: yes
          plugin_habits_from: 200
          plugin_habits_days: 14
          plugin_habits_facts: yes
          plugin_habits_charts: yes
          plugin_habits_trim: yes
          config_timezone: Asia/Kolkata

      # ── Isometric Calendar ───────────────────────────────────────────────────
      - name: Isometric Calendar
        uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          filename: metrics.isocalendar.svg
          base: ""
          plugin_isocalendar: yes
          plugin_isocalendar_duration: full-year
          config_timezone: Asia/Kolkata
```

> **Setup:** Go to **Settings → Developer Settings → Personal Access Tokens**, generate a token with `repo` + `read:user` scopes, then add it as a repository secret named `METRICS_TOKEN`.
